# ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๑ จังหวัดศรีสะเกษ เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง

ตามที่ได้มีพระราชกฤษฎีกาให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎร และคณะกรรมการการเลือกตั้งได้กำหนดให้วันที่ ๘ กุมภาพันธ์ ๒๕๖๙ เป็นวันเลือกตั้ง นั้น

บัดนี้ คณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๑ จังหวัดศรีสะเกษ ได้ดำเนินการรวมผลคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้งเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง ดังนี้

๑. จำนวนผู้มีสิทธิเลือกตั้ง
    ๑.๑ จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อผู้มีสิทธิเลือกตั้ง (หนึ่งแสนสามหมื่นสองพันเก้าร้อยยี่สิบเจ็ด ) ๑๓๒,๙๒๗ คน
    ๑.๒ จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน (เก้าหมื่นแปดพันเจ็ดร้อยเจ็ดสิบ ) ๙๘,๗๗๐ คน
๒. จำนวนบัตรเลือกตั้ง
    ๒.๑ จำนวนบัตรเลือกตั้งที่เขตเลือกตั้งได้รับจัดสรร (หนึ่งแสนสามหมื่นเจ็ดพันสามร้อยสามสิบสาม ) ๑๓๗,๓๓๓ บัตร
    ๒.๒ จำนวนบัตรเลือกตั้งที่ใช้ (เก้าหมื่นแปดพันเจ็ดร้อยเจ็ดสิบ ) ๙๘,๗๗๐ บัตร
    ๒.๓ จำนวนบัตรเลือกตั้งที่เหลือ (สามหมื่นแปดพันห้าร้อยหกสิบสาม) ๓๘,๕๖๓ บัตร
๓. จำนวนบัตรเลือกตั้งในหน่วยเลือกตั้ง
    ๓.๑ บัตรเลือกตั้งในหน่วยเลือกตั้ง (เก้าหมื่นหกร้อยหกสิบสาม) ๙๐,๖๖๓ บัตร
    ๓.๒ บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง (สิบสี่) ๑๔ บัตร
    ๓.๓ บัตรเลือกตั้งก่อนวันเลือกตั้งนอกเขตเลือกตั้งและนอกราชอาณาจักร (แปดพันเก้าสิบสาม) ๘,๐๙๓ บัตร
๓.๔ รวม (เก้าหมื่นแปดพันเจ็ดร้อยเจ็ดสิบ) ๙๘,๗๗๐ บัตร
