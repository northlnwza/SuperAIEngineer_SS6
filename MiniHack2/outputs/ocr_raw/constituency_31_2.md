# ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ 2 จังหวัดบุรีรัมย์ เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง

ตามที่ได้มีพระราชกฤษฎีกาให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎร และคณะกรรมการการเลือกตั้งได้กำหนดให้วันที่ 8 กุมภาพันธ์ พ.ศ. 2569 เป็นวันเลือกตั้ง นั้น

บัดนี้ คณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๒ จังหวัดบุรีรัมย์ ได้ดำเนินการรวมคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้งเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง ดังนี้

1. จำนวนผู้มีสิทธิเลือกตั้ง
    1.1 จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อผู้มีสิทธิเลือกตั้ง 127,640 คน (หนึ่งแสนสองหมื่นเจ็ดพันหกร้อยห้าสิบ)
    1.2 จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน 81,656 คน (แปดหมื่นหนึ่งพันหกร้อยห้าสิบหก)

2. จำนวนบัตรเลือกตั้ง
    2.1 จำนวนบัตรเลือกตั้งที่เขตเลือกตั้งได้รับการจัดสรร 123,257 บัตร (หนึ่งแสนสองหมื่นสามพันสองร้อยห้าสิบเจ็ด)
    2.2 จำนวนบัตรเลือกตั้งที่ใช้ 81,656 บัตร (แปดหมื่นหนึ่งพันหกร้อยห้าสิบหก)
    2.3 จำนวนบัตรเลือกตั้งที่เหลือ 41,601 บัตร (สี่หมื่นหนึ่งพันหกร้อยเอ็ด)

3. จำนวนบัตรเลือกตั้งที่ใช้นับคะแนน
    3.1 บัตรเลือกตั้งในหน่วยเลือกตั้ง 75,943 บัตร (เจ็ดหมื่นห้าพันเก้าร้อยสี่สิบสาม)
    3.2 บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง 0 บัตร (ศูนย์)
    3.3 บัตรเลือกตั้งก่อนวันเลือกตั้งนอกเขตเลือกตั้งและนอกราชอาณาจักร 5,713 บัตร (ห้าพันเจ็ดร้อยสิบสาม)
    3.4 รวม 81,656 บัตร (แปดหมื่นหนึ่งพันหกร้อยห้าสิบหก)

4. ผลการนับคะแนน
    4.1 บัตรดี 75,055 บัตร (เจ็ดหมื่นห้าพันห้าสิบห้า)
    4.2 บัตรเสีย 3,585 บัตร (สามพันห้าร้อยแปดสิบห้า)
    4.3 บัตรไม่เลือกผู้สมัครผู้ใด 3,016 บัตร (สามพันสิบหก)
