# ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๓ จังหวัดลพบุรี เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง

ตามที่ได้มีพระราชกฤษฎีกาให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎร และคณะกรรมการการเลือกตั้งได้กำหนดให้วันที่ ๘ เดือน กุมภาพันธ์ พ.ศ. ๒๕๖๙ เป็นวันเลือกตั้ง นั้น

บัดนี้ คณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๓ จังหวัดลพบุรี ได้ดำเนินการรวมผลคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้งเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง ดังนี้

## ๑. จำนวนผู้มีสิทธิเลือกตั้ง

๑.๑ จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อผู้มีสิทธิเลือกตั้ง ๑๔๕,๖๔๘ คน ( หนึ่งแสนสี่หมื่นห้าพันหกร้อยสี่สิบแปด)

๑.๒ จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน ๑๐๕,๙๓๐ คน (หนึ่งแสนห้าพันเก้าร้อยสามสิบ )

## ๒. จำนวนบัตรเลือกตั้ง

๒.๑ จำนวนบัตรเลือกตั้งที่เขตเลือกตั้งได้รับจัดสรร ๑๔๙,๕๐๐ บัตร (หนึ่งแสนสี่หมื่นเก้าพันห้าร้อย)

๒.๒ จำนวนบัตรเลือกตั้งที่ใช้ ๑๐๕,๙๒๙ คน (หนึ่งแสนห้าพันเก้าร้อยยี่สิบเก้า)

๒.๓ จำนวนบัตรเลือกตั้งที่เหลือ ๔๓,๕๗๑ บัตร (สี่หมื่นสามพันห้าร้อยเจ็ดสิบเอ็ด)

## ๓. จำนวนบัตรเลือกตั้งที่ใช้นับคะแนน

๓.๑ บัตรเลือกตั้งในหน่วยเลือกตั้ง ๑๐๑,๐๒๗ บัตร (หนึ่งแสนหนึ่งพันยี่สิบเจ็ด)

๓.๒ บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง ๒ บัตร ( สอง )
