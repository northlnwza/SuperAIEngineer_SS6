# ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๓ จังหวัดลพบุรี เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อ

ตามที่ได้มีพระราชกฤษฎีกาให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎร และคณะกรรมการ การเลือกตั้งได้กำหนดให้วันที่ ๘ เดือน กุมภาพันธ์ พ.ศ. ๒๕๖๙ เป็นวันเลือกตั้ง นั้น

บัดนี้ คณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๓ จังหวัดลพบุรี ได้ดำเนินการรวมผล คะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้งเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผลการนับ คะแนนสมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อ ดังนี้

๑. จำนวนผู้มีสิทธิเลือกตั้ง
๑.๑ จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อผู้มีสิทธิเลือกตั้ง ๑๔๕,๖๔๘ คน ( หนึ่งแสน สี่หมื่นห้าพันหกร้อยสี่สิบแปด)
๑.๒ จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน ๑๐๕,๙๓๐ คน (หนึ่งแสนห้าพันเก้าร้อยสามสิบ)

๒. จำนวนบัตรเลือกตั้ง
๒.๑ จำนวนบัตรเลือกตั้งที่เขตเลือกตั้งได้รับจัดสรร ๑๔๙,๕๐๑ บัตร (หนึ่งแสนสี่หมื่นเก้าพัน ห้าร้อยเอ็ด)
๒.๒ จำนวนบัตรเลือกตั้งที่ใช้ ๑๐๕,๙๓๐ คน (หนึ่งแสนห้าพันเก้าร้อยสามสิบ)
๒.๓ จำนวนบัตรเลือกตั้งที่เหลือ ๔๓,๕๗๑ บัตร (สี่หมื่นสามพันห้าร้อยเจ็ดสิบเอ็ด)

๓. จำนวนบัตรเลือกตั้งที่ใช้นับคะแนน
๓.๑ บัตรเลือกตั้งในหน่วยเลือกตั้ง ๑๐๑,๐๒๗ บัตร (หนึ่งแสนหนึ่งพันยี่สิบเจ็ด)
๓.๒ บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง ๒ บัตร ( สอง )
๓.๓ บัตรเลือกตั้งก่อนวันเลือกตั้งนอกเขตเลือกตั้งและนอกราชอาณาจักร ๔,๙๐๑ บัตร (สี่พัน เก้าร้อยเอ็ด)

๓.๔ รวม ๑๐๕,๙๓๐ คน (หนึ่งแสนห้าพันเก้าร้อยสามสิบ)

๔. ผลการนับคะแนน
๔.๑ บัตรดี -๙๖,๖๖๒ บัตร (เก้าหมื่นหกพันหกร้อยหกสิบสอง)
๔.๒ บัตรเสีย ๖,๓๗๑ บัตร (หกพันสามร้อยเจ็ดสิบเอ็ด)
๔.๓ บัตรไม่เลือกผู้สมัครผู้ใด ๒,๘๙๗ บัตร (สองพันแปดร้อยเก้าสิบเจ็ด)
