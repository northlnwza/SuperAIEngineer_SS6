# ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๓๓ กรุงเทพมหานคร เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง

ตามที่ได้มีพระราชกฤษฎีกายุบสภาผู้แทนราษฎร พ.ศ. ๒๕๖๘ มีผลบังคับใช้ตั้งแต่วันที่ ๑๒ ธันวาคม ๒๕๖๘ และคณะกรรมการการเลือกตั้งกำหนดให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎรเป็นการเลือกตั้งทั่วไป ในวันอาทิตย์ที่ ๘ กุมภาพันธ์ ๒๕๖๙ นั้น

บัดนี้ คณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๓๓ กรุงเทพมหานคร ได้ดำเนินการรวม ผลคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้งเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผลการ นับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง ดังนี้

๑. จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อ ๑๔๘,๑๖๔ คน ✓
    ผู้มีสิทธิเลือกตั้ง (หนึ่งแสนสี่หมื่นแปดพันหนึ่งร้อยหกสิบสี่)
๑.๒ จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน ๑๐๒,๖๒๘ คน (หนึ่งแสนสองพันหกร้อยยี่สิบแปด) ✓
๒. จำนวนบัตรเลือกตั้ง
    ๒.๑ จำนวนบัตรเลือกตั้งที่เขตเลือกตั้ง ๑๔๖,๕๘๐ บัตร ✓
    ๒.๒ จำนวนบัตรเลือกตั้งที่ใช้ ๑๐๒,๖๒๘ บัตร (หนึ่งแสนสองพันหกร้อยยี่สิบแปด) ✓
    ๒.๓ จำนวนบัตรเลือกตั้งที่เหลือ ๔๓,๙๕๒ บัตร (สี่หมื่นสามพันเก้าร้อยห้าสิบสอง) ✓
๓. จำนวนบัตรเลือกตั้งในหน่วยเลือกตั้ง
    ๓.๑ บัตรเลือกตั้งในหน่วยเลือกตั้ง ๙๗,๘๙๒ บัตร (เก้าหมื่นเจ็ดพันแปดร้อยเก้าสิบสอง) ✓
    ๓.๒ บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง ๑๕๗ บัตร (หนึ่งร้อยห้าสิบเจ็ด)
    ๓.๓ บัตรเลือกตั้งก่อนวันเลือกตั้งนอกเขตเลือกตั้งและนอกราชอาณาจักร ๔,๕๗๙ บัตร (สี่พันห้าร้อยเจ็ดสิบเก้า)
๔. ผลการนับคะแนน
    ๔.๑ บัตรดี ๙๔,๔๑๖ บัตร (เก้าหมื่นสี่พันสี่ร้อยสิบหก) ✓
    ๔.๒ บัตรเสีย ๒,๑๔๒ บัตร (สองพันหนึ่งร้อยสี่สิบสอง) ✓
    ๔.๓ บัตรไม่เลือกผู้สมัครผู้ใด ๖,๐๗๐ บัตร (หกพันเจ็ดสิบ) ✓
๕. จำนวนคะแนนที่ผู้สมัครรับเลือกตั้งแต่ละคนได้รับเรียงตามลำดับผู้ได้คะแนนสูงสุด
