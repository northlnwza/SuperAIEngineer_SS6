ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๑๑
จังหวัด กรุงเทพมหานคร
เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง

ตามที่ได้มีพระราชกฤษฎีกาให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎร และคณะกรรมการ
การเลือกตั้งได้กำหนดให้วันที่ ๘ เดือน กุมภาพันธ์ พ.ศ. ๒๕๖๙ เป็นวันเลือกตั้ง นั้น

บัดนี้ คณะกรรมการประจำเขตเลือกตั้งที่ ๑๑ จังหวัด กรุงเทพมหานคร ได้ดำเนินการรวมผล
คะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้งเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผล
การรวมคะแนนสมาชิกสภาผู้แทนราษฎรแบบแบ่งเขตเลือกตั้ง ดังนี้

๑. จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อผู้มีสิทธิเลือกตั้ง ๑๓๗,๓๑๙ คน (หนึ่งแสน
สามหมื่นเจ็ดพันสามร้อยสิบเก้าคน)
๑.๑ จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน ๑๐๐,๓๖๗ คน (หนึ่งแสนสามร้อยหกสิบเจ็ด)
๒. จำนวนบัตรเลือกตั้ง
๒.๑ จำนวนบัตรเลือกตั้งที่เขตเลือกตั้งได้รับจัดสรร ๑๓๙,๕๗๙ บัตร ( หนึ่งแสนสามหมื่น
เก้าพันห้าร้อยเจ็ดสิบเก้า )
๒.๒ จำนวนบัตรเลือกตั้งที่ใช้ ๑๐๐,๓๖๗ บัตร ( หนึ่งแสนสามร้อยหกสิบเจ็ด )
๒.๓ จำนวนบัตรเลือกตั้งที่เหลือ ๓๙,๒๑๒ บัตร ( สามหมื่นเก้าพันสองร้อยสิบสอง )
๓. บัตรเลือกตั้งในหน่วยเลือกตั้ง ๙๕,๗๓๘ บัตร ( เก้าหมื่นห้าพันเจ็ดร้อยสามสิบแปด )
๓.๒ บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง ๕๐ บัตร ( ห้าสิบ )
๓.๓ บัตรเลือกตั้งก่อนวันเลือกตั้งนอกเขตเลือกตั้งและนอกราชอาณาจักร ๔,๕๗๙ บัตร
(สี่พันห้าร้อยเจ็ดสิบเก้า)
๓.๔ รวม ๑๐๐,๓๖๗ บัตร (หนึ่งแสนสามร้อยหกสิบเจ็ด)

๔. ผลการนับคะแนน
๔.๑ บัตรดี ๙๔,๙๙๓ บัตร ( เก้าหมื่นสี่พันเก้าร้อยเก้าสิบสาม )
๔.๒ บัตรเสีย ๑,๒๐๗ บัตร ( หนึ่งพันสองร้อยเจ็ด )
๔.๓ บัตรที่ไม่เลือกผู้สมัครผู้ใด ๔,๑๖๗ บัตร ( สี่พันหนึ่งร้อยหกสิบเจ็ด )

๕. จำนวนคะแนนที่ผู้สมัครรับเลือกตั้งแต่ละคนได้รับเรียงตามลำดับผู้ได้คะแนนสูงสุด
