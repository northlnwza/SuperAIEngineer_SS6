# ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๑ จังหวัดศรีสะเกษ เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อ

ตามที่ได้มีพระราชกฤษฎีกาให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎร และคณะกรรมการ การเลือกตั้งได้กำหนดให้วันที่ ๘ กุมภาพันธ์ ๒๕๖๙ เป็นวันเลือกตั้ง นั้น

บัดนี้ คณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๑ จังหวัดศรีสะเกษ ได้ดำเนินการรวมผลคะแนน สมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผลการนับคะแนน สมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อ ดังนี้

๑. จำนวนผู้มีสิทธิเลือกตั้ง
    ๑.๑ จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อผู้มีสิทธิเลือกตั้ง (หนึ่งแสนสามหมื่นสองพันเก้าร้อยยี่สิบเจ็ด ) ๑๓๒,๙๒๗ คน
    ๑.๒ จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน (เก้าหมื่นแปดพันเจ็ดร้อยหกสิบแปด ) ๙๘,๗๖๘ คน
๒. จำนวนบัตรเลือกตั้ง
    ๒.๑ จำนวนบัตรเลือกตั้งที่เขตเลือกตั้งได้รับจัดสรร (หนึ่งแสนสามหมื่นเจ็ดพันสามสิบเอ็ด ) ๑๓๗,๓๓๑ บัตร
    ๒.๒ จำนวนบัตรเลือกตั้งที่ใช้ (เก้าหมื่นแปดพันเจ็ดร้อยหกสิบแปด ) ๙๘,๗๖๘ บัตร
    ๒.๓ จำนวนบัตรเลือกตั้งที่เหลือ (สามหมื่นแปดพันห้าร้อยหกสิบสาม) ๓๘,๕๖๓ บัตร
๓. จำนวนบัตรเลือกตั้งในหน่วยเลือกตั้ง
    ๓.๑ บัตรเลือกตั้งในหน่วยเลือกตั้ง (เก้าหมื่นหกร้อยหกสิบสาม) ๙๐,๖๖๓ บัตร
    ๓.๒ บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง (สิบสี่) ๑๔ บัตร
    ๓.๓ บัตรเลือกตั้งก่อนวันเลือกตั้งนอกเขตเลือกตั้งและนอกราชอาณาจักร (แปดพันเก้าสิบเอ็ด) ๘,๐๙๑ บัตร
    ๓.๔ รวม (เก้าหมื่นแปดพันเจ็ดร้อยหกสิบแปด) ๙๘,๗๖๘ บัตร
