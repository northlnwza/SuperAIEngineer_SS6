ส.ส. ๖/๑ (บช)

ประกาศคณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๒๒
จังหวัดกรุงเทพมหานคร
เรื่อง ผลการนับคะแนนสมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อ

ตามที่ได้มีพระราชกฤษฎีกาให้มีการเลือกตั้งสมาชิกสภาผู้แทนราษฎร และคณะกรรมการ
การเลือกตั้งได้กำหนดให้วันที่ ๘ เดือน กุมภาพันธ์ พ.ศ. ๒๕๖๙ เป็นวันเลือกตั้ง นั้น

บัดนี้ คณะกรรมการการเลือกตั้งประจำเขตเลือกตั้งที่ ๒๒ จังหวัดกรุงเทพมหานคร ได้ดำเนินการรวม
ผลคะแนนสมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อเสร็จสิ้นเป็นที่เรียบร้อยแล้ว ดังนั้น จึงขอประกาศผลการนับ
คะแนนสมาชิกสภาผู้แทนราษฎรแบบบัญชีรายชื่อ ดังนี้

๑. จำนวนผู้มีสิทธิเลือกตั้ง
๑.๑ จำนวนผู้มีสิทธิเลือกตั้งตามบัญชีรายชื่อผู้มีสิทธิเลือกตั้ง .. ๑๓๓,๖๑๒ ... คน (..หนึ่งแสน
สามหมื่นสามพันหกร้อยสิบสอง.)
๑.๒ จำนวนผู้มีสิทธิเลือกตั้งที่มาแสดงตน .. ๙๒,๓๐๑ ... คน (..เก้าหมื่นสองพัน
สามร้อยหนึ่ง ..)
๒. จำนวนบัตรเลือกตั้ง
๒.๑ จำนวนบัตรเลือกตั้งที่เขตเลือกตั้งได้รับจัดสรร .. ๑๓๗,๑๓๑ ... บัตร (..หนึ่งแสน
สามหมื่นเจ็ดพันหนึ่งร้อยสามสิบหนึ่ง ..)
๒.๒ จำนวนบัตรเลือกตั้งที่ใช้ .. ๙๒,๓๐๑ ... บัตร (..เก้าหมื่น/
สองพันสามร้อยหนึ่ง ..)
๒.๓ จำนวนบัตรเลือกตั้งที่เหลือ .. ๔๔,๘๓๑ ... บัตร (..สี่หมื่นสี่
พันแปดร้อยสามสิบเอ็ด ..)
๓. จำนวนบัตรเลือกตั้งที่ใช้นับคะแนน
๓.๑ บัตรเลือกตั้งในหน่วยเลือกตั้ง .. ๘๗,๙๖๙ ... บัตร (..แปดหมื่น
เจ็ดพันเก้าร้อยหกสิบเก้า)
๓.๒ บัตรเลือกตั้งก่อนวันเลือกตั้งในเขตเลือกตั้ง .. ๑๕๐ ... บัตร (..หนึ่งร้อยห้าสิบ)
(สี่พันหนึ่งร้อยแปดสิบสอง.)
๓.๓ บัตรเลือกตั้งก่อนวันเลือกตั้งนอกเขตเลือกตั้งและนอกราชอาณาจักร .. ๑,๑๘๒ ... บัตร
พันสามร้อยหนึ่ง ..)
๓.๔ รวม .. ๙๒,๓๐๑ ... บัตร (..เก้าหมื่นสอง
พันสามร้อยหนึ่ง ..)
๔. ผลการนับคะแนน
๔.๑ บัตรดี .. ๘๘,๒๘๓ ... บัตร (..แปดหมื่น
แปดพันสองร้อยแปดสิบสาม)
